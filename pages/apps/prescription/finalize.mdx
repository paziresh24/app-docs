# ثبت نهایی

![Finalize](/interfaces/prescription/finalize.png)

زمانی که پزشک در صفحه تجویز تصمیم به ثبت نهایی نسخه کرد لازم است کاربر به صفحه لیست مراجعین پذیرش24 به همراه اطلاعات نسخه شامل کدپیگیری و آیدی رکورد نوبت پذیرش24 برگردد.

#### هدایت کاربر

هدایت کابر پس ثبت نهایی باید با استفاده از متد `hamdast.intetnt` به همراه دیتای زیر انجام شود.

<div dir="ltr">

```sh npm2yarn
npm i @paziresh24/hamdast
```

</div>

```js
import { Hamdast, INTENTS } from "@paziresh24/hamdast";
const hamdast = new Hamdast();

function finalize() {
  // ...
  hamdast.intent(INTENTS.APPOINTMENTS_LIST, {
    prescription: {
      identifier: "XYZ",
      tracking_code: "123456789",
    },
  });
}
```

##### دیتای ارسالی

پس از هدایت کاربر به لیست بیماران پذیرش24 پیغامی به عنوان ثبت موفقیت آمیز نسخه نمایش داده خواهد شد.
