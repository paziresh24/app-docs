# بیمه ها

برای دسترسی به اطلاعات بیمه های پزشک برای ایجاد و نهایی سازی نسخه میتوانید از اطلاعات بیمه های وارد شده در پذیرش24 با پروتکل زیر دریافت کنید.

#### دریافت دسترسی و دریافت اطلاعات

<div dir="ltr">

```sh npm2yarn
npm i @paziresh24/hamdast
```

</div>

```js
import { Hamdast, PERMISSIONS } from "@paziresh24/hamdast";
const hamdast = new Hamdast();

// ...

const { data, status } = await hamdast.getUserData({
  requiredData: [PERMISSIONS.NAME_AND_FAMILY, PERMISSIONS.INSURANCES],
});

if (status === "granted") {
  return {
    insurances: data.insurances,
  };
}

// ...
```
